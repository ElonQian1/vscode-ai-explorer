## 🎨 新功能

### 浅层扫描（默认）
- 只显示当前目录的直接子项
- 速度极快
- 适合日常探索

### 双击下钻
- 双击文件夹 → 进入该文件夹
- 双击文件 → 打开编辑器

### 返回上级
- 点击 "⬆️ 返回上级" 按钮
- 或按 `Backspace` 键

### 自定义排除规则
```jsonc
// settings.json
{
  "filetreeBlueprint.excludes": [
    "**/.git/**",
    "**/target/**",
    "**/node_modules/**"
  ]
}
```
